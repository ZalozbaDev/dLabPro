Synopsis

Run DWT.

Description

Run FWT of signal idSignal using the desired transformation method (default is Daubechies D4) - see wvltype.

If idPitch is omitted pitch asynchronuous analysis is performed.

Parameters

   idSignal   data object containing signal to analyse
   idPitch   data object containing pitch information (maybe NULL)
   idReal   data object of transformed signal
   idImag   not used (NULL)

See also

   wvltype
