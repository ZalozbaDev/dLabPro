    Signature :    INT32 CDN3Stream_GetDNRecLen(CDN3Stream*, CDlpTable* lpiTable);
      Source :    dn3stream.c(1220)

Description

Calculates the length in bytes of a DNorm record corresponding to the component structure of lpiTable. The function takes into account that long integers have 32 bits in DNorm regardless of their actual platform dependent length.

Parameters

   lpiTable   A pointer to a CDlpTable instance

Return value

The length in bytes of a DNorm record for lpiTable
